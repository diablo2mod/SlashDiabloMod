Diablo II DC6 Write GIMP Plugin, for Windows
--------------------------------------------
By Havvoric

Installation
============
Copy gimp_dc6_w.exe to either the GIMP main plugin directory,
or your user specific one.

Usage
=====
1. Open/create your image in GIMP as normal. 
2. Convert the image to INDEXED, loading the necessary D2 palette.
3. Choose File, Save As...
4. Either
   a. Choose Diablo DC6 from File type list
or
   b. Ensure that you put ".dc6" after the filename
5. As normal, make sure that the directory selected is correct
